## DESIGN SPECIFICATIONS

### Animations & Interactions

**Global:**
- Smooth scroll behavior
- Subtle parallax on background elements
- Content fades/slides in on scroll

**Hero:**
- Background gradient slowly shifts (20s loop)
- Particles or grid lines animate subtly
- Text staggers in on load

**Cards:**
- Hover: Lift 4px, border glows gold
- Transition: 0.3s ease

**Buttons:**
- Hover: Scale 1.02, glow effect
- Active: Scale 0.98

**Stats:**
- Numbers count up when scrolled into view
- Duration: 2s, easing: ease-out

**Progress Bars:**
- Animate width on scroll into view
- Gradient fill moves subtly

### Mobile Responsiveness

**Breakpoints:**
- Desktop: 1200px+
- Tablet: 768px - 1199px
- Mobile: < 768px

**Mobile Adaptations:**
- Navigation: Hamburger menu
- Hero: Stack vertically, smaller text
- Cards: Single column stack
- Pricing: Stack with Growth first
- Sidebar nav: Hidden, accessible via menu
- Touch targets: Minimum 44px

### Performance Requirements
- Load time: < 3 seconds
- Lighthouse score: 90+ on all metrics
- Image optimization: WebP with fallbacks
- Lazy loading: Below-fold content
- No layout shift on load

### Accessibility
- WCAG AA compliance
- Keyboard navigation
- Screen reader friendly
- Alt text on all images
- Focus indicators visible
- Color contrast 4.5:1 minimum
